## Robot Arm Labs

This repo contains some labs from ECE470, a course on robot modelling and control. <br><br>

These labs consist of forward and inverse kinematics as well as gradient-based motion planning algorithms written in MATLAB, deployed on a real KUKA robot manipulator. <br><br>
